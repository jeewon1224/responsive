## Feature
- 미디어쿼리를 활용한 반응형 지원
- 슬라이더를 활용한 깔끔한 레이아웃의 베스트 섹션
- masonry를 적용한 이벤트 섹션
- Tweenmax를 활용한 동적 효과

## Using tool
- JavaScript 75.8%
- CSS 15.9%
- HTML 8.3%
